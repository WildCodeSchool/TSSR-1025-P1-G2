# 1. Utilisation de base
<span id="utilisation-de-base"></span>
Une fois le logiciel *John the ripper* installé sur notre machine, voici la première commande à effectuer pour récupérer le hash de notre fichier  compressé protéger par un mot de passe :

```
john-the-ripper.zip2john fichier1.zip > empreinte1.txt
```
